Daraxtlar va Graflar

Kirish

Daraxtlar va graflar - iyerarxik va tarmoqlangan ma'lumotlarni saqlash uchun mo'ljallangan murakkab ma'lumotlar tuzilmalari.

Bu tuzilmalar fayl tizimlari, tashkiliy tuzilmalar va ijtimoiy tarmoqlarni modellashtirish uchun juda qulay.

Daraxtlar

Daraxt - bu ierarxik ma'lumotlar tuzilmasi bo'lib, tugunlar va ularni bog'lovchi yo'llardan iborat.

Daraxt tugunini yaratish:


class TreeNode:
    def __init__(self, data):
        self.data = data
        self.children = []
        
    def add_child(self, child):
        self.children.append(child)
        
# Daraxt yaratish
root = TreeNode("Bosh direktor")
hr = TreeNode("HR bo'limi")
it = TreeNode("IT bo'limi")
root.add_child(hr)
root.add_child(it)

Ikkilik Daraxtlar

Ikkilik daraxt - har bir tugunda maksimal ikkita bola bo'lishi mumkin bo'lgan maxsus daraxt turi.

Ikkilik daraxt klassi:


class BinaryTreeNode:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

    def insert(self, data):
        if self.data > data:
            if self.left is None:
                self.left = BinaryTreeNode(data)
            else:
                self.left.insert(data)
        else:
            if self.right is None:
                self.right = BinaryTreeNode(data)
            else:
                self.right.insert(data)

Graflar

Graf - bu tugunlar (vertices) va qirralar (edges) to'plami. Graflar yo'naltirilgan yoki yo'naltirilmagan bo'lishi mumkin.

Graf yaratish:


class Graph:
    def __init__(self):
        self.vertices = {}
        
    def add_vertex(self, vertex):
        if vertex not in self.vertices:
            self.vertices[vertex] = []
            
    def add_edge(self, vertex1, vertex2):
        if vertex1 in self.vertices and vertex2 in self.vertices:
            self.vertices[vertex1].append(vertex2)
            self.vertices[vertex2].append(vertex1)  # Yo'naltirilmagan graf uchun

Aylanib o'tish

Daraxt va graflarda ma'lumotlarni aylanib o'tishning turli usullari mavjud.

Daraxt aylanib o'tish usullari:

  • Pre-order (Oldin-tartib)
  • In-order (Ichki-tartib)
  • Post-order (Keyin-tartib)

Graf aylanib o'tish usullari:

  • Breadth-First Search (BFS)
  • Depth-First Search (DFS)
